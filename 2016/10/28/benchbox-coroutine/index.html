2016-10-28
benchbox
benchbox  tbox  coroutine  context  switch  channel

Introduction

Benchbox is a benchmark testing utilities based on xmake and tbox.

Build

Please install xmake first: xmake

    $ xmake

$ xmake coroutine -n switch
